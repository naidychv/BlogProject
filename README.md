# Portfolio + Blog

Projekt strony portfolio z sekcją blogową.

## Opis

Strona składa się z dwóch głównych widoków:

- Strona główna (portfolio)
- Strona bloga

Portfolio zawiera informacje o autorze, umiejętnościach, projektach oraz formularz kontaktowy.

## Funkcjonalności

- Nawigacja między sekcjami
- Sekcja "O mnie"
- Lista umiejętności (HTML, CSS/SCSS, JS)
- Prezentacja projektów
- Formularz kontaktowy (HTML)
- Oddzielna strona bloga (blog.html)
- Dynamiczne ładowanie postów (script.js)

## Technologie

- HTML5
- SCSS
- JavaScript

## Struktura plików

- index.html – główna strona portfolio
- blog.html – strona bloga
- scss/main.css – stylowanie
- script.js – logika bloga
