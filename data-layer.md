# Warstwa danych

Projekt nie korzysta z bazy danych – dane są pobrane z plików JSON.

## Źródła danych

### index.html

Dane statyczne:

- informacje o autorze
- lista umiejętności
- projekty

### blog.html

Dynamiczna sekcja:

- <section id="posts"></section>

Dane są ładowane z plików JSON przez:

- script.js

## Operacje

- renderowanie postów w DOM
- manipulacja elementami HTML przez JS
